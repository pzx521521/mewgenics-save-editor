# Mewgenics Save File Modification Exploration

## Background
This work references the core code from https://mewgenics.kittyeditor.com/ to examine what the `blob` data stored in the `data` field of the `cats` table actually represents.

## Initial Idea
The original plan was to develop a save file editing tool.

## Consideration of Implementation
After some thought, a pure front-end implementation seemed to be a better approach.

## Purpose of This Record
Since there are already existing tools and related projects are not open source, this document only records the overall idea for reference.
